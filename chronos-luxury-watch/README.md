# Chronos Luxury Watch Website

A premium luxury watch website featuring cinematic scrolling animations and high-end UI interactions, designed to feel like a $5000 Apple product page.
 Features

### 🎯 **Advanced Interactions:**
- **3D Watch Viewer** - 360° interactive rotation with mouse drag and inertia
- **Custom Cursor** - Luxury trailing cursor with hover effects
- **Magnetic Buttons** - Buttons that move toward mouse cursor on hover
- **Ultra Premium Hero** - Cinematic intro with rotating watch and dynamic lighting

### 🎬 **Cinematic Animations:**
- **Engineering Story Scroll** - Sequential text reveals and gear animations based on scroll progress
- **Enhanced Scroll Storytelling** - Multi-layered animations triggered by scroll position
- **Smooth 60fps Animations** - All animations use requestAnimationFrame

### ✨ **Luxury Microinteractions:**
- **Card Hover Glow** - Shimmer effects on all interactive cards
- **Text Reveal Animations** - Staggered text appearances
- **Smooth Section Transitions** - Seamless transitions between sections
- **Enhanced Button Effects** - Ripple and shimmer animations

### 🚀 **Performance Optimizations:**
- **Advanced Lazy Loading** - IntersectionObserver with blur-to-sharp transitions
- **Optimized Animations** - will-change properties and efficient transforms
- **Image Performance** - Prevents layout shifts and enhances loading experience

### 🎨 **Premium Design System:**
- **Minimal Luxury Aesthetic** - Clean, dark theme with subtle lighting
- **Apple/Rolex Inspired** - High-end product page feel
- **Responsive Design** - Mobile-first approach with adaptive layouts
- **Cinematic Lighting** - Dynamic background effects and reflections

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Animations, Custom Properties)
- Vanilla JavaScript (ES6+)
- GSAP (for advanced animations)
- IntersectionObserver API
- requestAnimationFrame
